# FHE Medical Research Demo

A comprehensive demonstration of Fully Homomorphic Encryption (FHE) vs Partial Homomorphic Encryption for private medical research.

## Features

- **Two Encryption Schemes**: TenSEAL (CKKS FHE) vs Paillier (Partial HE)
- **Performance Comparison**: Speed benchmarks and capability analysis
- **Medical Data Simulation**: Realistic synthetic patient records
- **Beautiful Visualizations**: Comparative graphs and performance metrics
